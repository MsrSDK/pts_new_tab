# PTS_New_Tab

## 機能

- PTSのテーブル内のテキストをリンクに置き換えます。リンクをクリックすると新しいタブで開かれます。
- 対象ページ
  - [企業一覧]>[共同防除発注母体]
  - [企業一覧]>[パイロット]
  - [企業一覧]>[パイロット]>[パイロット一覧]
  - [案件]>[案件一覧]

## 使い方

### 初期設定

1. 拡張機能をインストール
1. 拡張機能を有効化
    - Chrome拡張機能管理ページ: `chrome://extensions/`

### 利用方法

1. 対象ページを開く
1. いずれかの方法で開発者ツールを開く
    - [ページ右クリック]>[検証]
    - Windows: [F12]キー or [Ctrl + Shift + I]
    - Mac: [Command + Option + I]

すでに開発者ツールが開かれているのにリンクに置き換えられない場合は以下の手順を実行してください。

1. ページをリロード
1. 開いている開発者ツールを閉じる
1. 開発者ツールを開く
1. ページを再度リロード
